Features & Capabilities

AI–Driven Symptom Analysis (Claude Opus 4.5)

Our chatbot uses Anthropic Claude Opus 4.5 via OpenRouter to analyse symptoms, classify urgency, and recommend the correct speciality.

OTP Verification (Fast2SMS)

Patient enters phone number → OTP sent through Fast2SMS Quick SMS API → verification ensures secure booking.

Calendly Slot Scheduling

Real-time slot availability, instant appointment creation, and automatic updates handled using Calendly APIs.

Booking, Reschedule & Cancel

Patients can book, move, or cancel slots. The chatbot interacts with Calendly to update schedules instantly.

SMS Confirmations

Fast2SMS sends booking confirmations, updates, and reminders automatically.

Complete Patient Summary

All details — name, phone, email, age, symptoms, doctor, slot time — shown in chat and sent via SMS.

Voice + Multilingual Chatbot

Supports English plus local language, and allows voice input for faster booking.

Operator Support (Optional)

Operators can review patient context, manage bookings, and assist manually if needed.

Complete Flow (Step-by-Step)

1

User opens chatbot

AI collects name, phone, email → sends OTP via Fast2SMS → verifies identity.

2

Symptom / Speciality Input

Claude Opus 4.5 analyses symptoms and chooses the right medical speciality.

3

Show slots

Bot fetches real-time slot availability from Calendly API and displays options.

4

Book Appointment

Slot confirmed → Calendly booking created → SMS confirmation sent instantly.

5

Reschedule & Cancel

Patients can shift or cancel bookings any time via chatbot commands.

6

Reminder Messages

Fast2SMS sends automated reminder SMS before appointment time.
